# PyramidMaker-C

**Pyramid Maker** – Kullanıcının girdiği yükseklik değerine göre konsolda piramit çizen interaktif bir C programı.

## Kullanım

1. Dosyayı derleyin:  
```bash
gcc pyramidMaker.c -o pyramidMaker -- //Kullandığınız derleyiciye göre
```
2. Uygulamayı çalıştırın.
3. Herhangi bir sayı deneyin.
